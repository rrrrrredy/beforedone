# v1.0.1 manual release evidence

This record closes the local/manual release boundary for BeforeDone v1.0.1.
Repository Actions were disabled before and after the manual release workflow,
which did not invoke them. This release does not claim GitHub OIDC build
provenance.

- Source commit and public `main` at release time:
  `cb7409bc2131c05d1cac06d05b38405162eabcaa`.
- Public tag: `v1.0.1`, pointing to the same commit.
- Published at: `2026-07-22T04:40:02Z`.

- Pages source: classic `gh-pages:/`, commit

## Quality and security gates

- `go test -count=1 ./...`, `go vet ./...`, and `go mod verify` passed on the
  release source. GoReleaser then repeated `go mod verify` and the complete Go
  suite from a clean clone of the final tag; the test hook passed in 6m49s.
- Canonical/Plugin Skill synchronization, distribution contract validation for
  version `1.0.1`, `goreleaser check`, `actionlint`, and `git diff --check`
  passed.
- Gitleaks scanned the worktree and all 12 reachable commits with no leaks.
- Release tools: Go 1.26.5 windows/amd64, GoReleaser 2.17.0, Syft 1.44.0,
  actionlint 1.7.7, and Gitleaks 8.30.1.
- The first isolated GoReleaser attempt stopped before testing or building
  because the newly assigned `GOTMPDIR` did not yet exist. No asset from that
  attempt was used. The directory was created and the release was rerun from
  the clean tag through all hooks and build stages.

## Explicitly outside this evidence boundary

This host did not execute the macOS, Linux, or Windows arm64 archives natively.
It has no native macOS/Linux test host or emulator, so cross-compilation,
metadata, archive, checksum, and SBOM inspection are not presented as native
runtime proof. The user accepted this as an explicit release exception.

At publication time, this record did not claim OpenAI Directory approval,
final media, YouTube publication, or a live Product Hunt listing. Those
optional launch activities were not completed and are not product capabilities.
